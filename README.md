## Installation


```bash
Database Setup
$ docker compose up
```

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

```bash
# API URL
http://localhost:3000/identify/

# Request Payload
{
	"phoneNumber": "1234517",
	"email":"abc+10@yopmail.com"
}
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
